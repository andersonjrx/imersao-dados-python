## 📜 Sobre o Projeto

O **Dashboard de Análise de Salários** é um aplicativo interativo desenvolvido para mapear e analisar remunerações no mercado de tecnologia, permitindo a exploração de filtros, cargos e cenários salariais de forma dinâmica.

Este projeto foi construído do zero aplicando conceitos de **Análise Exploratória de Dados**, **Limpeza de Dados**, **Data Storytelling** e **Business Intelligence (BI)** durante a **Imersão de Dados com Python**.

---

## ✨ Funcionalidades

* 🔍 **Exploração Dinâmica:** Filtre e analise as médias salariais de forma interativa por ano, senioridade, tipo de contrato e tamanho da empresa.
* 🧹 **Dados Tratados:** Pipeline completo de limpeza de dados, tratamento de valores nulos e preparação do dataset usando Pandas.
* 📈 **Data Storytelling:** Gráficos interativos projetados para contar a história por trás dos números de forma visual e intuitiva.
* 🖥️ **Dashboard Interativo:** Interface de usuário construída para transformar scripts Python em uma aplicação web acessível.
